Template Fields

What Every Trading Journal Template
Must Include

A proper trading journal template is not just a list of buy/sell prices. Here are the fields that actually drive performance improvement.

Trade Details

  • ✓ Date & time of entry/exit
  • ✓ Instrument / symbol name
  • ✓ Trade direction (Buy / Sell)
  • ✓ Entry price & exit price
  • ✓ Quantity / lot size
  • ✓ Total P&L (realised)

Setup & Strategy

  • ✓ Setup name or pattern
  • ✓ Reason for entry
  • ✓ Planned target price
  • ✓ Planned stop-loss price
  • ✓ Risk-to-reward ratio
  • ✓ Strategy or system name

Psychology

  • ✓ Emotional state before trade
  • ✓ Confidence level
  • ✓ Rule adherence (Yes/No)
  • ✓ Mistakes made (if any)
  • ✓ Post-trade notes / review
  • ✓ Chart screenshot

All of these fields are built into GreenJournal — structured, searchable, and automatically analysed.
